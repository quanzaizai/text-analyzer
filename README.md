# 文本分析小项目

## 项目说明
这是一个有python实现的文本分析小项目，这个项目代表了我第一次完成文本类数据的使用。

## 当前功能
- 从sample.txt文件中读取内容，随后进行文本打印
- 把一些长度等于1的文本数据给筛选掉
- 通过matplotlib实现文本数据的可视化，让数据更清晰

## 运行方式
```
uv run main.py
```

## 当前学习重点
- 第一次使用 jieba 库进行中文分词
- 使用列表推导式过滤停用词
- 使用 Counter 统计词频
- 使用 most_common() 获取高频词
- 使用 matplotlib 画词频柱状图
- 理解文本分析的基本流程：读取→分词→过滤→统计→可视化
