# 文本分析小项目

## 项目说明
这是一个用python实现的中文文本分析小项目，这个项目代表了我第一次完成文本类数据的使用。

## 当前功能
- 读取中文文本文件
- 使用 `jieba` 进行中文分词
- 过滤长度为 1 的词语
- 使用 `Counter` 统计词频
- 使用 `most_common()` 获取 Top5 高频词
- 使用 `matplotlib` 绘制词频柱状图
- 支持中文图表显示

## 运行方式
```
uv run main.py
```

## 当前学习重点
- 第一次使用 jieba 库进行中文分词
- 使用列表推导式过滤长度为 1 的词语
- 使用 Counter 统计词频
- 使用 most_common() 获取高频词
- 使用 matplotlib 画词频柱状图
- 理解文本分析的基本流程：读取→分词→过滤→统计→可视化
